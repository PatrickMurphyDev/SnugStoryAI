/specialconditions Endpoint Docs

Description

The /specialconditions endpoint allows for managing special condition data, including creating, retrieving, updating, and deleting special conditions.

Endpoint Schema

{
    "character_id": "ObjectId",
    "condition": "String",
    "description": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create a new special condition Sample
GET Get all special conditions Sample
GET /:id Get a special condition by ID Sample
PUT /:id Update a special condition by ID Sample
DELETE /:id Delete a special condition by ID Sample

Sample Payloads

POST /specialconditions Sample

{
    "character_id": "60d5f9e68f1b2c001c8e6fff",
    "condition": "Allergy",
    "description": "Character has an allergy to pollen"
}
