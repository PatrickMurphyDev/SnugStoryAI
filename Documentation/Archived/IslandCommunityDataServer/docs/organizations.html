/organizations Endpoint Docs

Description

The /organizations endpoint allows for managing organization data, including creating, retrieving, updating, and deleting organizations.

Endpoint Schema

{
    "name": "String",
    "type": "String",
    "address": "String",
    "description": "String",
    "building_id": "ObjectId"
}

Endpoint Methods

URL Method Description Sample
POST Create a new organization Sample
GET Get all organizations Sample
GET /:id Get an organization by ID Sample
PUT /:id Update an organization by ID Sample
DELETE /:id Delete an organization by ID Sample
