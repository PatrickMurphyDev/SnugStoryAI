/areas Endpoint Docs

Description

The /areas endpoint allows for managing area data, including creating, retrieving, updating, and deleting areas.

Endpoint Schema

{
    "island_id": "ObjectId",
    "name": "String",
    "type": "String",
    "income_level": "String",
    "description": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create a new area Sample
GET Get all areas Sample
GET /:id Get an area by ID Sample
PUT /:id Update an area by ID Sample
DELETE /:id Delete an area by ID Sample
