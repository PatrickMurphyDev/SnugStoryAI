/buildings Endpoint Docs

Description

The /buildings endpoint allows for managing building data, including creating, retrieving, updating, and deleting buildings.

Endpoint Schema

{
    "lot_id": "ObjectId",
    "name": "String",
    "type": "String",
    "address": "String",
    "description": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create a new building Sample
GET Get all buildings Sample
GET /:id Get a building by ID Sample
PUT /:id Update a building by ID Sample
DELETE /:id Delete a building by ID Sample
