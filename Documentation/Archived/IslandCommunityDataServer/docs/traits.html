/traits Endpoint Docs

Description

The /traits endpoint allows for managing trait data, including creating, retrieving, updating, and deleting traits.

Endpoint Schema

{
    "name": "String",
    "description": "String",
    "category": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create a new trait Sample
GET Get all traits Sample
GET /:id Get a trait by ID Sample
PUT /:id Update a trait by ID Sample
DELETE /:id Delete a trait by ID Sample

Sample Payloads

POST /traits Sample

{
    "name": "Bravery",
    "description": "The quality of being brave",
    "category": "psychological"
}

GET /traits Sample

[
    {
        "_id": "60d5fbe68f1b2c001c8e8fff",
        "name": "Bravery",

]

{
    "_id": "60d5fbe68f1b2c001c8e8fff",
