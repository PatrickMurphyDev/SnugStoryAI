/islands Endpoint Docs

Description

The /islands endpoint allows for managing island data, including creating, retrieving, updating, and deleting islands.

Endpoint Schema

{
    "name": "String",
    "location": "String",
    "description": "String",
    "areas": ["ObjectId"]
}

Endpoint Methods

URL Method Description Sample
POST Create a new island Sample
GET Get all islands Sample
GET /:id Get an island by ID Sample
PUT /:id Update an island by ID Sample
DELETE /:id Delete an island by ID Sample

Sample Payloads

POST /islands Sample

{
    "name": "Fantasy Island",
    "location": "Pacific Ocean",
    "description": "A beautiful tropical island"
}
