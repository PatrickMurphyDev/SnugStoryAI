/islandpropertylots Endpoint Docs

Description

The /islandpropertylots endpoint allows for managing island property lot data, including creating, retrieving, updating, and deleting island property lots.

Endpoint Schema

{
    "island_id": "ObjectId",
    "type": "String",
    "coordinates": "String",
    "description": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create a new island property lot Sample
GET Get all island property lots Sample
GET /:id Get an island property lot by ID Sample
PUT /:id Update an island property lot by ID Sample
DELETE /:id Delete an island property lot by ID Sample
