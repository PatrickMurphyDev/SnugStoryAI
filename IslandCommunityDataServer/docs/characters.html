/characters Endpoint Docs

Description

The /characters endpoint allows for managing character data, including creating, retrieving, updating, and deleting characters.

Endpoint Schema

{
    "name": "String",
    "age": "Number",
    "gender": "String",
    "sexualOrientation": "String",
    "occupation": "String",
    "job_id": "ObjectId",
    "residence_id": "ObjectId",
    "description": "String",
    "traits": ["ObjectId"],
    "resources_id": "ObjectId",
    "location_id": "ObjectId",
    "behavioral_patterns_id": "ObjectId",
    "special_conditions_id": "ObjectId"
}

Endpoint Methods

URL Method Description Sample
POST Create a new character Sample
GET Get all characters Sample
GET /:id Get a character by ID Sample
PUT /:id Update a character by ID Sample
DELETE /:id Delete a character by ID Sample
