/behavioralpatterns Endpoint Docs

Description

The /behavioralpatterns endpoint allows for managing behavioral pattern data, including creating, retrieving, updating, and deleting behavioral patterns.

Endpoint Schema

{
    "character_id": "ObjectId",
    "pattern": "String",
    "description": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create new behavioral patterns Sample
GET Get all behavioral patterns Sample
GET /:id Get behavioral patterns by ID Sample
PUT /:id Update behavioral patterns by ID Sample
DELETE /:id Delete behavioral patterns by ID Sample

Sample Payloads

POST /behavioralpatterns Sample

{
    "character_id": "60d5f9e68f1b2c001c8e6fff",
    "pattern": "Morning Jog",
    "description": "Character goes for a jog every morning"
