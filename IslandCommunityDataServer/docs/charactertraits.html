/charactertraits Endpoint Docs

Description

The /charactertraits endpoint allows for managing character trait data, including creating, retrieving, updating, and deleting character traits.

Endpoint Schema

{
    "character_id": "ObjectId",
    "trait_id": "ObjectId",
    "value": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create a new character trait Sample
GET Get all character traits Sample
GET /:id Get a character trait by ID Sample
PUT /:id Update a character trait by ID Sample
DELETE /:id Delete a character trait by ID Sample
