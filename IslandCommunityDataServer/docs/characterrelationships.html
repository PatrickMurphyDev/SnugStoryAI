/characterrelationships Endpoint Docs

Description

The /characterrelationships endpoint allows for managing character relationship data, including creating, retrieving, updating, and deleting character relationships.

Endpoint Schema

{
    "character_1_id": "ObjectId",
    "character_2_id": "ObjectId",
    "type": "String",
    "status": "String",
    "relationship_historical_event": ["ObjectId"]
}

Endpoint Methods

URL Method Description Sample
POST Create a new character relationship Sample
GET Get all character relationships Sample
GET /:id Get a character relationship by ID Sample
PUT /:id Update a character relationship by ID Sample
DELETE /:id Delete a character relationship by ID Sample
