/jobpositions Endpoint Docs

Description

The /jobpositions endpoint allows for managing job position data, including creating, retrieving, updating, and deleting job positions.

Endpoint Schema

{
    "organization_id": "ObjectId",
    "title": "String",
    "description": "String",
    "requirements": ["String"],
    "salary": "Number"
}

Endpoint Methods

URL Method Description Sample
POST Create a new job position Sample
GET Get all job positions Sample
GET /:id Get a job position by ID Sample
PUT /:id Update a job position by ID Sample
DELETE /:id Delete a job position by ID Sample
