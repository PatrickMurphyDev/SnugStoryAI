/location Endpoint Docs

Description

The /location endpoint allows for managing location data, including creating, retrieving, updating, and deleting locations.

Endpoint Schema

{
    "name": "String",
    "type": "String",
    "coordinates": "String",
    "description": "String"
}

Endpoint Methods

URL Method Description Sample
POST Create a new location Sample
GET Get all locations Sample
GET /:id Get a location by ID Sample
PUT /:id Update a location by ID Sample
DELETE /:id Delete a location by ID Sample

Sample Payloads

POST /location Sample

{
    "name": "Central Park",
    "type": "park",
    "coordinates": "40.785091, -73.968285",
    "description": "A large public park in New York City"
}
