/relationshipevents Endpoint Docs

Description

The /relationshipevents endpoint allows for managing relationship event data, including creating, retrieving, updating, and deleting relationship events.

Endpoint Schema

{
    "character_id": "ObjectId",
    "event": "String",
    "description": "String",
    "parent_stage_id": "ObjectId",
    "next_stage_ids": ["ObjectId"]
}

Endpoint Methods

URL Method Description Sample
POST Create a new relationship event Sample
GET Get all relationship events Sample
GET /:id Get a relationship event by ID Sample
PUT /:id Update a relationship event by ID Sample
DELETE /:id Delete a relationship event by ID Sample
